## 开发工具请使用(VSCode) ##
1. 插件: eslint
2.  文件>首选项>设置,打开VSCode配置文件,添加如下配置
```json
{
    "files.autoSave": "off",
    "eslint.validate": [
        "javascript",
        "javascriptreact",
        "vue-html",
        {
        "language": "vue",
        "autoFix": true
        }
    ],
    "eslint.run": "onSave",
    "eslint.autoFixOnSave": true
}
```
